# 📦 Release Guide — Better Sound.Maker v1.0.0

This guide provides information on how to download, install, and update the **Better Sound.Maker** desktop application.

---

## 📥 Download

You can find the latest stable installer in the **Releases** section of this GitHub repository.

- **File name:** `Better-Sound-Maker-Setup-1.0.0.exe`
- **Platform:** Windows 10 / 11 (x64)
- **Size:** ~95 MB

---

## 🛠️ Installation Instructions

1. **Download the Installer:** Get the `.exe` file from the latest release.
2. **Run the Setup:** Double-click the downloaded file.
3. **SmartScreen Notice:** Since the app is currently self-signed, Windows might show a "Windows protected your PC" popup. 
   - Click on **"More info"**.
   - Click on **"Run anyway"**.
4. **Automatic Launch:** The application will install itself and launch automatically. A shortcut will be added to your desktop and start menu.

---

## 🔄 Updating

The application currently does not feature an auto-updater. To update to a newer version:
1. Download the new version of the installer.
2. Run it — it will automatically replace the old version while keeping your local projects and settings safe.

---

## ⚠️ Requirements & Troubleshooting

### Requirements
- **OS:** Windows 10 or 11.
- **DCS World:** Compatible with version 2.9 and higher.
- **Storage:** At least 200MB of free space for the app and cached asset lists.

### Antivirus Issues
Some antivirus software (like Windows Defender or Avast) might flag the installer as "unknown" because it's a new, community-made tool. This is a false positive. If the installer is blocked, you may need to temporarily disable your antivirus or add an exclusion for the app.

---

## 🎧 First Steps in the App

1. **Language Selection:** Choose your preferred language (English/French).
2. **Mascot Tutorial:** Follow "Tasky", our mascot, for a quick 2-minute tour of the features.
3. **Asset Library:** On your first run, the app will load the default DCS asset list to get you started immediately.
