# Update 1.0.8 — Migration & Features Restored

This update marks the migration of the project to its new GitHub home, preparing the way for a future organization.

## Patch Notes

*   **New Migration Home**: The app is now fully linked to the new GitHub account `better-dcs`. All project links, documentation, and the community presets API now point to the new location.
*   **Auto-Update Restored**: Reactivated the automatic update checking system to ensure all users stay on the latest version.
*   **Community Presets Enabled**: Re-enabled the community presets browser, allowing you to browse and apply shared sound configurations again.
*   **License Display Fix**: Fixed an issue where the Apache 2.0 license wouldn't load in the Credits page. It now supports multiple branch fallbacks (`main`/`master`) for reliability.
*   **Consistency Fixes**: Squashed several small bugs where the old version number (1.0.7) or old repository links were still visible in some menus or locales.
*   **Build Optimization**: Enhanced the internal packaging configuration for smoother maintenance and updates.
