# 🚀 Better Sound.Maker — Version 1.0.9

We are excited to announce the release of version **1.0.9**! This update marks a significant milestone with the official migration to our new GitHub repository and the return of essential features.

## 🛠️ Major Changes

### 🏠 New GitHub Repository & Auto-Update
The project has officially moved to **FreeProject089**. Thanks to this migration:
- **Auto-Update is back!** The application can once again check for and download the latest updates automatically.
- All internal links, documentation, and download references have been updated to point to the new repository.

### 🍱 Community Presets
The link for **Community Presets** has been fixed and now points to the repository maintained by **BetterDCS**, ensuring you always have access to the latest community presets.

### 🎨 Visual Identity
- Updated the application logo with the new **BSM.png** design.
- Refreshed icons and branding throughout the interface (Credits, Sidebar, Favicon).

### 📦 Technical Fixes
- Fixed a critical bug that prevented building via `electron-builder` due to an invalid package name.
- Optimized project metadata.

---

## 📥 How to Update?
If you are already using version 1.0.8, the auto-updater should prompt you for the update at the next launch. Otherwise, you can download the new version directly from [GitHub](https://github.com/FreeProject089/Better-Sound.Maker/releases).
