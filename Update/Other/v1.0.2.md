# What's New in v1.0.2 🛠️

Welcome back! We've made some important tweaks to **Better Sound.Maker**.

### 🔄 True Auto-Updates
The app now monitors our GitHub repository for new releases using `electron-updater`. Instead of prompting you to download from your browser, it will silently download the latest update in the background, without interrupting your flow! 

Once the download is ready, you will be prompted to restart, or you can just let it apply smoothly the next time you close your PC or app.

### 💨 Faster Application Exit
We've cleaned up the old manual update check delays so when you close Better Sound.Maker, it closes instantly without lingering in the background.
