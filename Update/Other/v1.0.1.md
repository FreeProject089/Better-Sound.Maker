# What's New in v1.0.1 🎧

Welcome back! We've made some exciting updates to **Better Sound.Maker**:

### 🔄 Auto-Updater
The app now monitors our GitHub repository for new releases. You'll be notified at launch or when closing the app if a new version is available, ensuring you always have the latest features and bug fixes.

### 🤖 Meet Tasky!
Our mascot finally has a name: **Tasky**! Tasky will walk you through the application during the onboarding tutorial.

### ✍️ Dynamic Tutorials
Tutorial text now appears with a **lightning-fast typing animation**, making the onboarding experience feel more modern and interactive.

### 📄 Release Tracker
You can now view all past update notes directly in the app. All notes are stored in the `/Update` folder for easy reference.
